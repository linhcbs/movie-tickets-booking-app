# cinema-booking-system
Một web app đơn giản để đặt vé xem phim.

## Hướng dẫn chạy
### Chạy local

1. Cài đặt NodeJS.
2. Clone repo này về.
3. Mở terminal và `cd` vào folder `app` của repo.
4. Chạy command `npm install`
5. Chạy command `node server.js` (hoặc `nodemon server.js` nếu đã tải nodemon để không cần restart lại server thủ công khi save file)
6. Vào browser và truy cập vào `http://localhost:[PORT]/` (PORT mặc định là 3000)
